 # WAZNE

Trzeba doinstalować bibliotekę cx_Oracle do pythona

````

python -m pip install cx_Oracle --upgrade pip

````

Trzeba utworzyć folder "libraries" i skopiować do niego zawartość linka (Basic Package dla swojego OSu):

````
https://www.oracle.com/database/technologies/instant-client/downloads.html

````

Do pliku config.py w zmiennych username, password wpisać login i hasło do swojej bazy danych!!!
